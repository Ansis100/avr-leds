# ATMega328P LED chaser

## Prerequisites
Arch - `avr-libc` & `avr-gcc`
Ubuntu - `avr-libc` & `gcc-avr`

## Compilation
Command `make` compiles, links files and tries to flash to ATMega328P.
If flashing is not needed, then the flashing error can be ignored.
